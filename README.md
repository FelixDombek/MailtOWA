# MailtOWA

### Chrome Extension to open mailto: links with Outlook Web Application (OWA)

... the missing bit of functionality that Microsoft should have provided?

### Usage

* Install in Chrome
* Click any `mailto:` link and compose directly in OWA
* Profit
